## 김재민 파트 branch

### 👨‍💻 코드 설명 및 팁

**1. `QVideoWidget`과 `QMediaPlayer`를 분리한 이유**
* **QVideoWidget:** 껍데기(TV 화면)입니다. 화면에 영상을 띄워주는 역할만 합니다.
* **QMediaPlayer:** 리모컨(셋톱박스)입니다. 재생, 정지, 파일 로딩 등 기능을 담당합니다.
* 코드를 보시면 `self.media_player.setVideoOutput(self.video_widget)` 부분에서 이 둘을 연결해주고 있습니다.

**2. `Qt.UserRole`이 뭔가요? (비밀 주머니)**
* 리스트 화면(`file_list`)에는 '파일 이름'만 보여주지만, 영상을 재생하려면 'C:\Users\...\video.mp4' 같은 **전체 경로**가 필요합니다.
* 화면에는 안 보이지만 데이터를 숨겨두기 위해 사용하는 공간이 `Qt.UserRole`입니다. `setData`로 숨기고, `data`로 꺼내 쓸 수 있습니다.

**3. 백엔드 팀원과 협업하는 방법**
* 코드 중간에 `add_dummy_data` 함수가 있습니다. 지금은 제가 가짜 데이터를 넣어놨지만, 나중에 백엔드 팀원이 DB에서 데이터를 가져오는 함수를 만들면 그 부분을 교체하면 됩니다.
* **"야, 내가 리스트에 보여줄 데이터 받을 자리는 만들어놨어. 너는 DB에서 `[[파일명, 경로, 태그], ...]` 형태로 리스트 주는 함수만 짜오면 돼."** 라고 말하면 아주 멋있어 보일 겁니다.
