# 영상 분류기 (Video Classifier)

### 이태훈, 여인영, 김재민

## 프로젝트 개요

이 프로젝트는 지정된 폴더 내의 모든 비디오 파일을 찾아 목록으로 만들고, 사용자가 태그를 추가하거나 파일을 검색할 수 있도록 돕는 간단한 데스크톱 애플리케이션입니다.

## 주요 기능

- **폴더 선택**: 사용자는 비디오 파일을 검색할 최상위 폴더를 선택할 수 있습니다.
- **비디오 스캔 및 정보 추출**:
  - 선택된 폴더와 그 하위 폴더들을 재귀적으로 탐색하여 일반적인 비디오 파일을 찾습니다.
  - 각 비디오 파일에 대해 파일명, 비디오 길이, 전체 경로 등의 메타데이터를 추출합니다.
- **데이터 저장**:
  - 모든 비디오 정보는 `videos.json` 파일에 저장되어 프로그램 재시작 시에도 유지됩니다.
- **비동기 처리**: 파일 검색 작업은 백그라운드에서 실행되어, 작업 중에도 UI가 멈추지 않습니다.
- **리스트 상호작용**:
  - **비디오 재생**: `파일명` 셀을 더블 클릭하여 시스템의 기본 플레이어로 비디오를 즉시 재생합니다.
  - **폴더 열기**: `경로` 셀을 더블 클릭하여 비디오 파일이 위치한 폴더를 엽니다.
  - **태그 편집**: `태그` 셀을 더블 클릭하여 각 비디오에 대한 태그를 쉽게 추가하고 수정합니다.
- **검색 및 필터링**:
  - `전체`, `파일명`, `확장자`, `태그` 등 다양한 기준으로 비디오 목록을 실시간으로 검색하고 필터링할 수 있습니다.

## 설치 및 실행 방법

### 1. 사전 요구사항

- Python 3.x 버전이 설치되어 있어야 합니다.

### 3. 프로그램 실행

다음 명령어를 사용하여 프로그램을 실행합니다.

```bash
uv run video_classifier.py
```

## 사용 방법

1.  **폴더 선택**: 프로그램이 실행되면 `Select Folder` 버튼을 클릭하여 비디오 파일이 있는 폴더를 선택합니다.
2.  **파일 스캔**: 폴더를 선택하면 자동으로 파일 스캔이 시작되며, 스캔이 완료되면 비디오 목록이 `파일명`, `길이`, `태그`, `경로` 순으로 나타납니다.
3.  **상호작용**:
    - **비디오 재생**: 목록에서 재생하고 싶은 비디오의 `파일명` 부분을 더블 클릭하면 영상이 재생됩니다.
    - **폴더 위치 열기**: `경로` 부분을 더블 클릭하면 해당 파일이 있는 폴더가 열립니다.
    - **태그 추가/수정**: `태그` 부분을 더블 클릭하면 팝업 창이 나타나며, 원하는 태그를 입력하고 `Save` 버튼을 누릅니다.
4.  **검색**: 상단의 검색창에 키워드를 입력하여 목록을 필터링할 수 있습니다. 검색창 왼쪽의 드롭다운 메뉴에서 검색 유형을 변경할 수 있습니다.
